# libusb source

This directory vendors libusb 1.0.29 from https://github.com/libusb/libusb at
commit `15a7ebb4d426c5ce196684347d2b7cafad862626`. It is built statically into the
Sensor Probe JNI library for Android USB fd wrapping and USB transfers.
